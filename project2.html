project rekenmachine

Dit project betreft een zelfgemaakte rekenmachine. Met behulp van JavaScript en CSS heb ik deze eenvoudige maar functionele rekenmachine ontwikkeld. Het stelt gebruikers in staat om basisberekeningen uit te voeren, waaronder optellen, aftrekken, vermenigvuldigen en delen. Als extra functionaliteit heb ik een geschiedenisfunctie toegevoegd waarmee gebruikers hun vorige berekeningen kunnen bekijken.
