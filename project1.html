Project goot het in de bak

Gooi het in de bak" is een project waarbij ik samen met een groep een website heb ontwikkeld om mensen te motiveren tot recyclen. We hebben gebruik gemaakt van Node.js, JavaScript, HTML en CSS. Mijn focus lag voornamelijk op het ontwerpen van de homepagina. Ik heb ook een interactieve quiz toegevoegd met behulp van JavaScript om gebruikers bewust te maken van het belang van recyclen.
